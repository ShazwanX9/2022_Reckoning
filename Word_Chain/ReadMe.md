# Word Chain
___
    Words are joined if the last letter of one word 
    and the first letter of another word are the same.
___

```
    Required:
        list - array of words

    Examples:
        arr = ["make", "eat", "tame"]
        milipede_word(arr) => ["make", "eat", "tame"]
        is_milipede_word(arr) => True

        arr = ["make", "eat", "tame", "buy"]
        milipede_word(arr) => ["make", "eat", "tame"]
        is_milipede_word(arr) => False
```
