# Bijective Binary

___
    Maximal number of regions obtained
    by joining n points around a circle
    by straight lines.

    Also number of regions in 4-space
    formed by n-1 hyperplanes.

    ```
    First six points:
        1 -> 1
        2 -> 2
        3 -> 4
        4 -> 8
        5 -> 16
        6 -> 31
        .... ...
    ```
___

    Required:
        number of points around a circle

    Example:
        region_in_circle(test) => 74519
