# Coprime to N
___
In mathematics, two integers a and b are coprime, 
relatively prime or mutually prime if the only 
positive integer that is a divisor of both of them is 1.
___

```
    Required:
        :param n:   ->    int     - n
        :return:    ->    list    - all the coprime to n

    Examples:
        coprimes(100) == [
            1, 3, 7, 9, 11, 
            13, 17, 19, 21, 
            23, 27, 29, 31, 
            33, 37, 39, 41, 
            43, 47, 49, 51, 
            53, 57, 59, 61, 
            63, 67, 69, 71, 
            73, 77, 79, 81, 
            83, 87, 89, 91, 
            93, 97, 99
        ]
```
