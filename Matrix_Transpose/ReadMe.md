# Transpose of a matrix

___
    What the program does is 
    Take a matrix, 
    Return a new matrix where the columns and rows of the original are swapped
___

    Required:
        :param matrix: original matrix to be transposed

    Example:
        transpose([[1, 2, 3], [4, 5, 6], [7, 8, 9]])
                            => [[1, 4, 7], [2, 5, 8], [3, 6, 9]]
